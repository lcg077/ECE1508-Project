# Traffic Flow Prediction

Graph Neural Networks for Traffic Flow Prediction on METR-LA dataset.

## Models
- Baseline: LSTM
- Main model: STGCN (Spatial-Temporal Graph Convolutional Network)

## Dataset
METR-LA: 207 traffic sensors in Los Angeles.

## Requirements
pip install -r requirements.txt

## How to Run
Open ECE1508Project.ipynb in Google Colab and run all cells.
